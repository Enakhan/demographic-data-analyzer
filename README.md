# Demographic Data Analyzer

This is a Python project that analyzes demographic data using the **Pandas** library. The dataset contains census data from the 1994 Census database.

## 📊 Project Overview
The goal of this project is to answer specific questions about the dataset, such as:
* How many people of each race are represented?
* What is the average age of men?
* What is the percentage of people who have a Bachelor's degree?
* What percentage of people with advanced education make more than 50K?
* What country has the highest percentage of people that earn >50K?

## 🛠️ Tools Used
* **Python 3**
* **Pandas** (for data manipulation)
* **Jupyter Notebook** (for exploratory data analysis)
